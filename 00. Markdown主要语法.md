## 2. 段落格式
### 2.1 换行
#### 2.1.1 在段落后面加两个空格表示换行。    
>君不见  
>黄河之水天上来

#### 2.1.2 在段落后使用一个*空行*表示换行并开始一个新段落。

>高堂明镜悲白发

### 2.2 字体
>*斜体文本*  
>_斜体文本_  
>**粗体文本**  
>__粗体文本__  
>***粗斜体文本***  
>___粗斜体文本___  

### 2.3 分割线
在一行中用三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西。星号或是减号中间可以插入空格。
> 段落一
> ****
> 段落二
> 
> ----
> 段落三
> _____
  
### 2.4 删除线
在文字的两端加上两个波浪线 `~~` 即可。
> ~~再苦一苦百姓，骂名我来担~~
>~~我说另请高明吧~~

### 2.7 高亮

## 3. 列表
### 3.1 无序列表

无序列表使用星号`*`、加号`+`或是减号`-`作为列表标记，这些标记后面要添加一个空格，然后再填写内容。
> * 第一项
> * 第二项
> + 第三项
> + 第四项
> - 第五项
> - 第六项

### 3.2 有序列表
数字并加上`.`号来表示有序列表。
> 1.第一  
> 2.第二  
> 3.第三  

### 3.3 列表嵌套
在标号前添加空格即可。

> 1.第一章
> * 1.1第一节
>   * 1.1.1第一部分
>     * 1.1.1.1 第一题

~~~python
#拓展欧几里得算法，密码学最基础的算法。
def extendgcd(a,b):#返回一个集合：(x,y,d)满足ax+by=d
    if b == 0:
        return (1,0,a)
    else:
        (x,y,d)=extendgcd(b,a%b)
        x,y=y,(x-(a//b)*y)
        return (x,y,d)
print(extendgcd(12,36))
print(extendgcd(56,3))
print(extendgcd(4589,393))
~~~

## 11. CSS增强

高级 Markdown 编辑器支持内联 HTML 和 CSS，这使得 Markdown 的格式化能力变得无比强大。

在 Markdown 中嵌入 **CSS 样式** 示例：

test {style="color:red;border-bottom: 10px double black"}

<style>
  .wavy-text {
    text-decoration: underline wavy red;   /* 单属性搞定 */
    text-underline-offset: 0.2em;         /* 线与字距离 */
  }
</style>

<span class="wavy-text">我下面就是波浪线～</span>


<style>
    /* 核心语法标记 - 极简版 */
    .s { color: red;border-bottom: 5px double red; padding-bottom: 1px; }     /* 主语-双线 */
    .p { border-bottom: 2px solid blue; padding-bottom: 1px; }    /* 谓语-实线 */
    .o {
    text-decoration: underline wavy red;   /* 单属性搞定 */
    text-underline-offset: 0.2em;         /* 线与字距离 */
    }


    .attr { background-color: #fff3cd; padding: 0 2px; }          /* 定语-黄底 */
    .adv { background-color: #d1ecf1; padding: 0 2px; }           /* 状语-蓝底 */  
    .comp { background-color: #f8d7da; padding: 0 2px; }          /* 补语-粉底 */

</style>

<span class="s">The young girl</span> 

<span class="p">happily showed</span> 

<span class="o">her friends</span> 
<span class="angle comp">the pictures</span> 

<span class="square adv">yesterday</span>。

---

### 1. 基本原理：HTML 标签与 `style` 属性

*   **HTML（超文本标记语言）**：是构建网页的基石。它使用 **“标签”** （如 `<div>`, `<span>`, `<p>`）来定义内容的结构。
*   **CSS（层叠样式表）**：是控制网页表现的语言。它决定这些 HTML 内容**如何被呈现**（颜色、大小、布局等）。
*   **`style` 属性**：是 HTML 标签的一个通用属性，用于给**单个标签**添加独特的 CSS 样式。其语法是 `style="property: value;"`。

在您的例子中，Yanknote 很可能在渲染时，自动将您的文本用了一个简单的 HTML 标签（如 `<span>`）包裹起来，并加上了这个 `style` 属性，最终在浏览器中生成的代码是这样的：

```html
<span style="color: red; border-bottom: 10px double black;">测试内容</span>
```

---

### 2. 样式属性详解

您的 `style` 声明里包含了两个 CSS 属性，用分号 `;` 分隔：

1.  `color: red;`
    *   **属性（Property）**： `color`
    *   **值（Value）**： `red`
    *   **作用**： 将文本的**字体颜色**设置为红色。您也可以使用十六进制码（如 `#ff0000`）、RGB 值（如 `rgb(255, 0, 0)`）或其他颜色名称。

2.  `border-bottom: 10px double black;`
    *   **属性（Property）**： `border-bottom`。这是一个**简写属性**，用于一次性设置下边框的多个特性。
    *   **值（Value）**： 包含了三个子值：
        *   `10px`： 边框的**宽度**。`px` 是像素单位。
        *   `double`： 边框的**样式**。`double` 表示双线。其他常见值还有 `solid`（实线）、`dashed`（虚线）、`dotted`（点线）等。
        *   `black`： 边框的**颜色**。
    *   **作用**： 为元素的底部添加一个 **10 像素粗的、黑色的、双线样式的** 边框。正是这个属性产生了您看到的“双下划线”效果。

---

### 3. 为什么这不是真正的“下划线”？

传统的“下划线”通常使用 CSS 属性 `text-decoration: underline;` 来实现。

但您使用的 `border-bottom` 更加灵活和强大：
*   **控制力强**：您可以独立控制这条“线”的**粗细**、**样式**、**颜色**，而 `text-decoration` 的限制较多。
*   **效果丰富**：通过设置 `double`, `dashed` 等样式，您可以轻松实现双线、虚线等传统下划线无法实现的效果。
*   **定位灵活**：您还可以用 `padding-bottom` 来控制文本和这条线之间的距离。

**简单来说：您用“底部边框”模拟并超越了下划线的效果。**

### 4. 还能怎么玩？（更多CSS属性示例）

既然可以用 `style` 属性，那几乎所有的CSS文本样式都可以用在您的笔记里。例如：

| 样式代码 | 效果说明 |
| :--- | :--- |
| `{style="background-color: yellow;"}` | 黄色高亮背景 |
| `{style="font-weight: bold; color: blue;"}` | **加粗的蓝色文字** |
| `{style="font-style: italic;"}` | *斜体文字* |
| `{style="font-size: 20px;"}` | 增大字号 |
| `{style="border: 2px solid red; padding: 5px;"}` | 给文字加一个红色实线的**盒子** |

### 总结

您发现的 `{style="..."}` 语法是一个强大的“后门”，它让 Markdown 的格式化能力直接提升到了网页级别。通过嵌入 CSS，您可以精确地控制笔记的视觉外观，实现各种富文本效果，这对于做**视觉化学习笔记**（比如用不同颜色和线条区分语法成分）来说，是一个极其有用的功能！

如果您想系统地了解还有什么属性可用，可以搜索“CSS text properties”或“CSS border properties”，会发现一片新天地。
